# openclaw-context

Base de conhecimento do sistema OpenClaw — Alexandre Aguiar
MENSURA Engenharia | MIA Engenharia | PCS Engenharia

## Como usar

Adicione arquivos (`.md`, `.pdf`, `.txt`) em qualquer pasta.
O sistema processa automaticamente a cada 30 minutos.

## Estrutura

- `institucional/` → apresentações e posicionamento das marcas
- `metodologia/` → Lean Construction, Last Planner, referências técnicas
- `projetos/` → documentos por projeto ativo
- `referencias/` → livros e artigos de referência
